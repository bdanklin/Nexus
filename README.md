# Elexus Hub

A Lightweight wrapper for the Nexus Hub API, written in Elixir.

> > > > > > > d046bb61a3e3d5fff104ea57bd966df3d812f353

## Installation

The package can be installed by adding `elexus_hub` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:elexus_hub, "~> 0.1.0"}
  ]
end
```
